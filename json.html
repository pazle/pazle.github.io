Json (JavaScript Object Notation)

This is designed to interface with APIs aminly web and sockets since it is now one of the most used ways to transfer data among different languages. This json string is turned to a krein dictionary and back.

Notice from this module.

Safely json syntax is required since krein passes the json string as a normal variable in its interpreter and later returns a dictianary so if a syntax error, the whole program breaks.

Json with comments is supported and others that include.

  • Arrays
  • Strings
  • Numbers
  • Objects

Table of attributes.

parse

Converts a json string to a krein dictionary.

jsonize

Converts a krein dictionary to a json string.

JSON.parse( strjson )

Takes one argument and it should be a json string in utf-8 encoding.

JSON.jsonize( dict {} )

Takes one argument and it should be a krein dictionary. It will then return a json string but not formatted at all.

import JSON let settings = ` { "fontSize": 15, "color": "red", "queries": [1020, 720, 480, 320, 220], }` x = JSON.parse(settings) x['font_size'] = 17; toJson = JSON.jsonize(x) print(toJson)
