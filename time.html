Time

This module is designed to include time functions in your code, it uses your systime time and local timezone. More functionality is to be provided in coming updates because alot is missing.

Notice from this module.

UTC time is not included, it bases on your system time and local timezone, if you want to adjust to other timezones, this could be impossible for now, but updates are coming soon. Another option is using an API for web applications.

  • Time Station API
  • World Time API
  • Sunrise-Sunset API

Use the curl library to make requests and parse there returned data using the JSON library. boom the application is done.

Table of attributes.

now

Returns today's date and time as a string. *

epoch

Returns the unix time or epoch since computers started counting time.

sleep

Pauses the executing thread for the specified time.

Date

Returns an object with time properties like hours to years. *

time.now( )

Takes no argument, but in coming updates a timezone or timestamp will be used as an argument.

time.epoch( )

No argument is used but still expect one in coming updates.

time.sleep( int secs )

One integer argument is used as seconds for the duration to sleep the thread.

time.Date( )

Also here an argument is not used but expect one in coming updates, this could be a timezone, timestamp or epoch.
