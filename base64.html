Base64 encoding

This modules provides functions to deal with encoding and decoding to the famously used base64 strings and back to bytes.

Notice from this module.

The generated strings are normally longer than the given data may be by 30% but it accomplishes the work.

Table of attributes.

encode

Encodes given bytes data to a base64 string.

decode

Decodes a base64 string back to byte data.

encode_url

Encodes given bytes data to a base64 string that is safe for urls and filenames.

decode_url

Decodes a base64 string back to byte data that is safe to be used for urls and filenames.

Base64.encode( bytes )

Takes one argument that is of bytes datatype.

Base64.decode( str )

Takes one argument that is a string with base64 characters.

Base64.encode_url( bytes )

Takes one argument that is of bytes datatype.

Base64.decode_url( str )

Takes one argument that is a string with base64 characters.

import base64 txt = 'hello world!' x = base64.encode(txt.encode()) print(x) //aGVsbG8gd29ybGQh print(base64.decode(x)) //hello world!
