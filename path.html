Path

This module deals with paths on any operating system, you can tell if a path is a file or directory. This module goes hand in hand with the file module.

Notice from this module.

I prefer / to \ to be used as a path separator on any operating system. Remember on Windows Pazle supports both / and \ like 'app\bin\main.rn' and 'app/bin/main.rn'.
