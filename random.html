Random

This is designed to generate random numbers and data mathematically for Kreinlang using math alogorithms and formulars to accomplish the job. It inspired from Javascript Random Js library from npm. .

Notice from this module.

This library's random generated data is not safe to be used for security purposes. For security or cryptographic uses, you can wrap around its generated data and derive out some algorithms to generate safer numbers and more.

Table of attributes.

shuffle

Shuffles array data to a random different order.

random

Generates a double number that can be used from the krein interpreter. The result is less than 1 or 1.

pick

Gets a random item from a provided array or strings since they are arrays too.

string

Genrates a random string ranging from a_zA_Z including 0-1 and you can add more characters to fetch from the string.

stringPool

Generates the string from the provided characters and a specified length or the pool length.

sample

Chooses elements from an array randomy of a specified length.

bool

Generates a random boolean value true or false. Chances for each ocurring is equal no percentage chances favour one.

real

Produces a random real number from the specified range.

integer

Produces a random integer number from the specified range.

random.shuffle( [1, 2, 3, 4] )

Takes one argument and it should be an array. Procuces a shuffled list like [3, 1, 4, 2].

random.random( )

Uses no argument generating a number between 0.0000000001 and 1. You can them multiply it to units you want tens, hundreds or etc.

random.pick( [1, 2, 3, 4, 'a', 'b'] )

Takes one argument and it should be an array.

random.string( int length, Opstr pool )

Takes one argument or two, one is the length that should be an integer and argument two is optional, its a string of chars that you can add to the string pool that the string is generated from.

random.stringPool( str pool, int len )

Here you enter your string charaters to generate from the string. This can be tweaked around to get random hex values by only providing 0-9a-fA-F, characters.

Takes two arguments a string and an integer for length of generated string.

random.sample( [1, 2, 3, 4], int length )

Remember its generated array is in ascending order of the element position although random elements are gathered.

Takes two arguments an array and an integer for length of array to generate.

random.bool( )

No argument is used and chances of true or false resulting are equal.

random.real( int min, Opint max)

A real number is generated according to your max and min integers or 0.

random.integer( int min, Opint max)

An int number is generated according to your max and min integers or 0.
