Variables

Krein variables can store any datatype, cool! There isn't any restriction to using them just like python variables. You can use a simple let statement just to decorate your code which can't even be seen by the interpreter.

let x = 'Lionel Messi' y = 'Christiano Ronaldo' teams = ['Man U', 'Barcelona'] goals = 5

Functions

Krein functions are simply variables that return data after performing a task assigned to them when they are called. If you assigned to them params, well you have to pass to them args respective to the params during calling them.

Some functions are already imported to the working environment..

def greet(name) { print(f'How are you {name}?') } greet('Carol') // How are you Carol?

Functions can return fellow functions and recurse too.

Objects

These can also be called types, you create custom datatypes with this. All variables and functions created in this class scope turn into the class's attributes after it is initiated.

defobject Animal { def this(name, kind) { self.name = name self.kind = kind self.__repr__ = name } } animals = [ Animal('cow', 'domestic'), Animal('warthog', 'wild'), Animal('whale', 'marine'), ] for i in animals print(f'{i.name}\t.... {i.kind}')

this is our constructing functions.

Switch Statements

Rare in some scripted languages but we have it in krein.

x = 5 switch x { case 2 { print('first') } case 5 { print('second') break } default { print('default') } }

Try catch Statements

This one is used to catch errors during runtime in code execution, it is not perfect for now but has the language evolves, this statement will become much more better and faster.

y = 9 try { // remember z is not defined print(y + z) } try { print(z) } catch e { print(e) }
