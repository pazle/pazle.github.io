def > functions

Pazle functions are procedures that are used to do repetitive tasks and those that are to be used in later execution.

These are variables that even return data but only after they have done a task.

Creating a function.

We use the def keyword then the function name, the parameters followed by the function's scope.

// this one returns none // it greets the parameter name def greet(name) { print(f'How are you {name}?') } // calling the function greet('Rogers') // >>> How are you Rogers?

Returning data from a function.

Use the return keyword followed by an expression or the function will return a null value.

// returning the sum // of its params def sum(x, y) { return x + y } add = sum(5, 2) //>> 7

Anonymous functions.

Use the return keyword followed by an expression or the function will return a null value.

// sum function add = ?(x, y) { return x + y } add(5, 2) //>> 7

Default parameters.

Pazle functions support default parameters and alot is gonna be added to this feature of syntax.

// normal params def minus(x, y){ return x-y } minus(5, 2) // >> 2 // function with default param def greet(name='Louis'){ return f'How are you {name}?' } greet() // >> How are you Louis? greet('Phoebe') // >> How are you Phoebe?

Note

Default parameters come last after normal parameters.
