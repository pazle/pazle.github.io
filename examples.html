Making, opening and handline files.

We are going to check if a file exists and if not we create it, then write to it some data. Check if the operation was successful by reading the file. And then close the file. Also find the file size.

import file, path filename = 'test.txt' if (!path.exists(filename)) file.write(filename, '') stream = file.open(filename, 'w') user_input = readline('What to write to the file\n: ') stream.write(user_input) stream.close() size = file.size(filename) if (size) { stream = file.open(filename, 'r') print(stream.read()) print(f'size {size} bytes') stream.close() }

Guessing game.

We are creating a guessing game with a set answer as lion where you require userinput for the guess as it decrements your scores which will be already as 5 since you have five chances. Such programs can be binded to a gui making it a full game for production.

guess = 'lion' qn = 'What is the bravest animal?' def prove(ans) { if (ans == guess) return true return false } def play() { scores = 5 while (scores) { user = readline('Guess: ') if prove(user) return ['You win', scores] scores -= 1 } return ['You lose', scores] } print(qn) x = play() print(f' {x[0]') print(f'scores {x[1]}')

import pazle ag = pazle.argv if (length(ag) > 2) { if ag[1] == 'greet' print(f'How are you {ag[2]}?') elif ag[1] == 'hey' print(f'Hey {ag[2]}?') else print(f'For now I can only greet.') } else { print('expects a name and options from [greet | hey]') }
