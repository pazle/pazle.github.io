Array data > arrays

This datatype can also be called a container for it is used to store other datatypes of any kind supported by Pazle.

This is still an object with attributes for modifying it but still a more low level robustic module is provided to make coding better.

Creating an array.

Arrays can be created with both [] & () delimeters and items are separated by , commas. They are very easy to use.

let letters = ['a','b','c','d','e'] nums = [1, 2, 3, 4, 5, 6] mixed = ['a', 'b', 1, 2.5, letters, [1, 2]] // you can put any datatype in arrays

Accessing items from arrays or Indexing arrays.

We can accesses data from arrays using there index and also change the data according to the index. Index start from 0 which is like the pointer of the first item in the array and they go on incrementing.

letters = ['a','b','c','d','e'] a = letters[0] b = letters[1] // changing data 'c' to 1.5 letters[2] = 1.5 // ['a','b',1.5,'d','e']

To get multiple data from the array we use arrays module, its easy to use.

import arrays l = ['a','b','c','d','e'] arrays.fetch(2, 4) // ['c','d','e'] arrays.lfetch(3) // ['a', 'b', 'c']

Operation on arrays.

Concatenating arrays is also easy, reversing and the rest. Follow the example. More arrays functions are in the attributes table.

import arrays as a a = [1,2,3] b = [4,5,6] c = a.concate(a, b) // [1,2,3,4,5,6] a.reverse(c) // [6,5,4,3,2,1]

Table of attributes.

The arrays module has low level functions that work with arrays.

pop

[ ].pop(int-opt index)
Pops off an item from the array either the indexed one or the last one.

append

[ ].append(any)
appends an item to the array.

clear

[ ].clear()
Empties the array.

insert

[ ].insert(int index, data)
Inserts data at the given index.

concate

arrays.concate(array*)
Concates the arrays and returns a single one.

reverse

arrays.reverse(array)
Reverses the array.

copy

arrays.copy(array)
Returns a copy of the array.

fetch

arrays.fetch(array, int min, int-opt max)
Returns array items ranging from index min to max or from index min to the last item.

get

arrays.get(array, int max)
Returns array items ranging from 0 to max.

Notice

This module functions are faster that custom functions that can do the same so I recommend using them.
