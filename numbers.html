Pazle numbers can be both integers and float which is meant to simplify work but still in the background you are dealing with over 7 types of numbers that include.

    But double & long are the most favoured for the Pazle runtime. Negative numbers are also supported.

    Operations on numbers.

    Just like all programming languages, you have to be able to add, subtract, divide and multipy numbers which is possible here. using the operands +, -, *, /, %

    // adding numbers sum = 5 + 4.5 // 9.5 // subtracting numbers minus = 7 - 8 // 1 // diving x = 4 / 2 // 2 // multiplying minus = 7 * 2 // 14 // remainder rem = 7 % 2 // 1

    Mathematic on numbers

    You have to use the math module to do all kinds of math functions you want to carryout straigth from raound-off to trigonometry and many more.

    import math // rounding to 2 dp x = math.round(34.4372, 2) // 34.44 // cosine of 90 y = math.cos(90) // -0.448074

    Note

    Its only numbers operations that are supported in the Pazle interpreter's runtime evaluation and the rest are made possible with functions that somehow helps in execution speed.
