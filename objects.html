defobject > objects

Every datatype used in Pazle is an object which makes Pazle an object oriented programming language. Were each data has its own attributes that define it more.

You can build custom types with objects and give them your own attributes which could be functions or variables.

Creating an object.

We use the defobject keyword then the object name, if yo are to inherit any other object, pass it as a parameter followed by the object's scope.

// worker's type defobject Workers { def this(name, salary, sector){ self.name = name self.salary = salary self.sector = sector // how the object should look on logging self.repr() } def repr(){ self.__repr__ = self.name } } tech = [Workers('Jim', 50_000, 'Management'), Workers('Cate', 25_000, 'Teaching')] // [Jim, Cate]

Function attributes.

We can do more with objects by giving functions..

// from previous example defobject Workers { def details(){ return [self.name, self.salary, self.sector] } } worker1 = tech = Workers('Cate', 25_000, 'Accountance') print(worker1.name) // Cate worker1.salary = 50_000 worker1.details() // ['Cate', 30000, 'Accountance']

this function.

this function is the constructor of the object, it is executed on calling the type which creates the new object.

defobject TempDB { def this(){ self.db = []; print('created temp database') } def insert(val){ self.db.append(val) } } db1 = TempDB() // created temp database db1.insert(1) db1.insert(2) print(db1.db) // [1, 2]

Note

Default parameters come last after normal parameters.
