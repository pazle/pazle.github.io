String data > strings

This is by far the most useful datatype as it is the safest to be transmitted over networks like base64 and json strings. It this best for filling files and many more.

Strings are then supported full in the Pazle interpreter handling them, interacting and modifying the strings is by far made easy with simple functions. Encoding of strings is supported too.

Remember Pazle uses utf-8 encoding for its code syntax making it support imojis embedded straight in your code and encoded characters too for different languages like Japanese and Korean characters.

Creating a string.

Strings can be created with both '' and "" as this is because Pazle has not datatypes likes char with are mostly used in compiled programming languages. But remember the interpreter is still using the D and C datatypes in the background to make all this translation possible for the registers or CPU I guess.

let name = 'Pazle' // formatted string formatted = f'How are you {name}'

Unicode in Pazle strings.

Unicode like hex-unicode is supported straight from the source code including IMOJIS.

// hex-unicode smile imoji let smile = '\xae\xf3\xbf' imoji = ''

Operation on strings.

Table of attributes.

The strings module has low level functions that work with strings.

lower

strings.lower(str)
Returns the lowercase if the string.

upper

strings.upper(str)
Returns the uppercase if the string.

replace

strings.replace(str, str chars1, str chars2)
Replaces the chars1 to chars2 in str.

hex

strings.hex(str)
Converts and return string to hexidecimal string.

hexToStr

strings.hexToStr(str hex)
Converts hexidecimal string to a utf-8 string.

trim

strings.trim(str opt)
Trims string from the end following the second param of trims the newline.

concate

strings.concate(str *)
Adds multiple strings.

times

strings.times(str, int)
Multiplies the string by the specified sec int parameter.

strip

strings.strip(str, str chars)
Strips off chars from both the start and end of the string.

lstrip

strings.lstrip(str, str chars)
Strips off chars from the start of the string.

rstrip

strings.rstrip(str, str chars)
Strips off chars from the end of the string.

indexOf

strings.indexOf(str, str char)
Returns the index number of the first specified char in the string.

splitLines

strings.splitLines(str)
Returns an array of strings after spliting the string at newline characters.

split

strings.split(str, str opt-chars)
Returns an array after spliting the string at the chars in the string.

Notice

This module functions are faster that custom functions that can do the same so I recommend using them.
