Console

This module is very essential because it deals with the interpreter core functions that are used to check datatypes states print there info to the terminal. It is even imported directly to the runtime for you.

List of functions in the console module.

Since console is imported for you, you are not expected to re-import it unless otherwise but even you will still be accessing the same pointer in memory.

    Using console.

    console.array( object )

    Converts the provided data to an array.

    console.attr( object )

    List all attributes of the provided data.

    console.bytes( object )

    Converts the provided data to bytes data.

    console.delattr( object, str attr )

    Removes an attribute from the provided object.

    console.exit( )

    Stops the calling runtime thread from executing.

    console.getattr( object, str attr )

    Gets the value of the attribute of the provided object.

    console.iter( object )

    Converts the provided object into iterable dataif possible or supported.

    console.length( object )

    Returns the length of the provided object.

    console.next( iter object )

    Increments tto the next value of the provided iterable object.

    console.number( object )

    Converts the provided data into a number or 0/zeror if not supported.

    console.print( *objects )

    Logs the given objects __str__ attribute on the terminal or stdio.output .

    console.range( int min, int max )

    Creates a range iterable from min to max.

    console.readline( str )

    Gets user input from terminal.

    console.setattr( object, str attr, object value )

    Sets the attribute of the provided object to a new value.

    console.string( object )

    Converts the given object to a string.

    console.super( object, object )

    Adds attributes of the inherited object to inheriting object.

    console.type( object )

    Returns the type of the provided object as a string.
