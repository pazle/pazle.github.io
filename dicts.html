Dict data > dicts

This datatype mimics a dictionary where a word is followed by its defination. It is safe to use and can handle alot of data. It can easily be converted to a json string for usage in other programs and runtimes.

Creating a dictionary.

Dictionaries require a key with a corresponding value, during storage there is no need to follow the order they were add to the dict since we access there data using the key.

import dicts let settings = { 'fullscreen': true, 'font-size': 15, 'accent-color': '#ff000090', 'theme': 'dark' } print(settings) // { "font-size": 15, "theme": "dark", "fullscreen": true, "accent-color": "#ff000090" }

Accessing items from dictionaries.

We use a key get the value and also modify the data at that key. A specific key can only be one in a dictionary or else the value will be over written.

// a key should always be a string let l = {'a': 1, 'b': 2, 'c': 3, 'd': 4} l['a'] // 1

A key should always be a string or it can be written in this fomat.

def saver { print('saved file') } def opener { print('open file ...') } let op = { save: saver, open: opener } // { "save": saver (method), "open": opener (method) }

Changing dictionaries values.

Access the key and directly assign to it a value of use the set function in the dicts module.

import dicts as dt let sets = { 'fullscreen': true, 'font-size': 15, 'accent-color': '#ff000090', 'theme': 'dark' } // { "font-size": 15, "theme": "dark", "fullscreen": true, "accent-color": "#ff000090" } sets['font-size'] = 12 dt.set(sets, 'theme', 'light') print(sets) // { "font-size": 12, "theme": "light", "fullscreen": true, "accent-color": "#ff000090" }

Table of attributes.

The dicts module has low level functions that work with Pazle dictionaries.

copy

dicts.copy(dict)
Returns a new copy of the dict.

get

dicts.get(dict, string key)
Returns the value of the given key.

has

dicts.has(dict, string key)
Returns true if the given key is in the dict.

keys

dicts.keys(dict)
Returns all of the dict keys.

pop

dicts.pop(dict)
Removes an item from the dictionary following the given key and returns its value.

set

dicts.set(dict, string key, any value)
Sets the dict key to new data or creates a new key with its given value.

values

dicts.values(dict)
Returns all of the dict values.
